# Password Checker

This is a clean, modern password strength checker built with HTML, CSS, and JavaScript.  
What it does is that it evaluates the strength of a password based on length, presence of uppercase letters, numbers, and special characters, and displays a live strength indicator.

## Features

- Real-time password strength feedback
- Color-coded strength bar (Weak, Medium, Strong, Very Strong)
- Simple, responsive design

## How to Use

### Run Locally

1. Clone or download this repository.  
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, Safari).  
3. Type a password in the input field to see the strength indicator update live.

### Deployment

This project is designed to be hosted on platforms like GitHub Pages for free, with live access on the web.
